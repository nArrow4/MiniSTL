# MiniSTL

基于C++11的小型STL

在这里你可以学到基础数据结构、算法、现代C++、Traits技法、SFINAE技法等等

# Tutorial

[记录一下学习过程中的笔记](./docs/Tutorial.md)

# Dependence

* Win11 + WSL2 + Ubuntu20.04
* cmake 3.16.3
* g++ 9.4.0
* gtest

# Build & Run

```sh
git clone https://github.com/nArrow4/MiniSTL.git
mkdir build && cd build
cmake ..
make -j4
./${test_demo_name}
```
